Python §4

Python §4 is a small programming language, designed for the fourth chapter of the textbook Structure and Interpretation of Computer Programs, JavaScript Adaptation (SICP JS).

What names are predeclared in Python §4?

On the right, you see all predeclared names of Python §3, in alphabetical order. Click on a name to see how it is defined and used. They come in these groups:

  • MISC: Miscellaneous constants and functions
  • MATH: Mathematical constants and functions
  • LINKED LISTS: Support for linked lists
  • PAIR MUTATORS: Mutating pairs
  • LISTS: Support for lists
  • STREAM: Support for streams
  • MCE: Support for the meta-circular evaluator

What can you do in Python §4?

You can use all features of Python §3 and the three functions that are introduced in chapter 4 of the textbook, given in MCE.
